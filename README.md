## PrismAI - AI Bias Detector

## Overview
PrismAI is a simple web app that detects bias in text using AI. It helps users find unfair or biased language and suggests a more neutral version.

## Features
- Detects bias level (Low, Medium, High)
- Identifies type of bias
- Gives a short explanation
- Suggests a neutral version
- Shows confidence score
- Clean and easy interface
- Basic voice input support

## Tech Stack
- Python  
- Streamlit  
- Google Gemini API  

## How to Run
- Install requirements: `pip install -r requirements.txt`  
- Run the app: `streamlit run app.py`  

## Use Cases
- Improve fair communication  
- Check bias in content  
- Learning and awareness  
- Useful for students and creators
